# KRAPI Style Guide

## Colors

### Light Theme

- `--text`: `#0f1208` - Text
- `--background`: `#f0f4e6` - Background
- `--primary`: `#545d28` - Borders, main CTAs
- `--secondary`: `#85cc91` - Secondary buttons
- `--accent`: `#48ad77` - Links, highlights

### Dark Theme

- `--text`: `#f4f7ed` - Text
- `--background`: `#15190b` - Background
- `--primary`: `#ced7a2` - Borders, main CTAs
- `--secondary`: `#337a3f` - Secondary buttons
- `--accent`: `#52b781` - Links, highlights

## Usage

- **Backgrounds**: `bg-background`
- **Borders**: `border-primary`
- **Primary buttons**: `bg-primary text-background`
- **Secondary buttons**: `bg-secondary text-text`
- **Links**: `text-accent`
- **Text**: `text-text`

## Typography

- **Font**: IBM Plex Mono (monospace)
- **Sizes**: `text-sm`, `text-base`, `text-xl`, `text-2xl`, `text-3xl`, `text-4xl`, `text-5xl`
- **Weights**: `font-normal` (400), `font-bold` (700)
